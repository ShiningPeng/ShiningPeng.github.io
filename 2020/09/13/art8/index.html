vue tab切换功能

  • 这个功能用vue的双向数据绑定实现起来就很简单，结合指令能达到比较好的效果

    <!-- tab -->
<view class="treaty-header">
  <button
    class="treaty-btn btn-sse"
    :class="treatyActive === 'sse'?'treaty-active':''"
    @click="treatyActive = 'sse'"
  >SSE</button>
  <button
    class="treaty-btn btn-eth"
    :class="treatyActive === 'eth'?'treaty-active':''"
    @click="treatyActive = 'eth'"
  >ETH</button>
</view>

<!-- tab对应的内容 -->
<view class="treaty-content">
<view class="treaty-left content">
  <view v-for="(item, index) in (treatyActive==='sse'?list:list3)" :key="{index}">
    <index-treaty-item :rate="item.rate" :title="item.title" :index="index"></index-treaty-item>
  </view>
</view>
<view class="treaty-middle content">
  <view v-for="(item, index) in (treatyActive==='sse'?list1:list4)" :key="{index}">
    <index-treaty-item :rate="item.rate" :title="item.title" :index="index"></index-treaty-item>
  </view>
</view>
<view class="treaty-right content">
  <view v-for="(item, index) in (treatyActive==='sse'?list2:list5)" :key="{index}">
    <index-treaty-item :rate="item.rate" :title="item.title" :index="index"></index-treaty-item>
  </view>
</view>
</view>
    data() {
  return {
    treatyActive:'sse'
  }
}
    .treaty-active {
  color: #ffffff !important;
  background: linear-gradient(135deg, #8990ed 0%, #757cff 100%) !important;
}

  • 首先就是类名绑定，这里使用的是三元运算符，当treatyActive的值等于该按钮设定的值时就给这个元素加上类名，当值改变时就删除了这个类名，这个类名的特别样式需要先写好，让人知道现在处于这个tab。然后在这个tab被点击时，修改treatyActive的值，tab会立即改变，同时tab内容部分也会进行条件判断来展示不同的内容

周日的下厨时光

  • 前两天室友买了电饭煲，周六晚上也计划好了第二天也就是周日做个红烧肉，懒人红烧肉，用电饭煲做。于是一早我俩就到附近菜市场买了一斤二两猪肉，猪肉28一斤，价格还是挺高的，我还以为这些天已经降下来了。然后去菜市场旁边的超市买所需的调料，因为做这个需要调料挺多的。买了八角，桂皮，干辣椒，盐，香叶，老抽，生姜，蒜，大葱，啤酒，东北大米。

电饭煲懒人红烧肉教程

  • 买好猪肉叫热心摊主切块，但是摊主好像会错意了，切成块的肉被扔进绞肉机绞了一下，成肉片。。。
  • 猪肉冷水下锅焯水，撇去血沫，喜欢血沫的同学可以不撇，等到肉被煮成白色就可出锅了
  • 将焯好水的肉放入电饭煲
  • 加入八角，桂皮[，干辣椒]，香叶，老抽（适量），适量盐，生姜几片，蒜几颗，大葱切段，加入一罐啤酒，肉多的话可以多放
  • 起锅烧油，哦不电饭煲通电开始煮，等个四五十分钟即可，在等待过程中可以搅拌一下，让调料与肉亲密接触，产生情愫。
  • 时间到，出锅，呈上米饭就可开吃了。（因为就一个锅，所以是先煮好米饭，然后挖出放入碗中，加盖保温）
  • 成品图片（略，卖相不好and没装盘）
  • 下面进入技术总结：
    • 老抽要适量，放的老抽太少，导致上色不深，颜色较浅
    • 香叶几片就行不用太多
    • 使用冰糖，因为我有白砂糖，所以在超市就没买冰糖，这也可能是上色失败的原因之一
  • 技术总结完毕

ending

  • 放假做菜放松一下，让紧张的神经放松下来，心情愉悦，明天好上班。
